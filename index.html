Get Started Locally

Ensure you have Python 3.12+ installed, then execute the following commands in your shell:

PowerShell / Terminal 
# 1. Clone the repository
git clone https://github.com/darshannaik019/chess_py.git
cd chess_py

# 2. Install Pygame and Python-chess dependencies
pip install -r requirements.txt

# 3. Launch the game application
python src/main.py
